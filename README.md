# Wekinator-gesture-projekt


Dette er mit wekinator projekt, med forskellige input gestures, som giver forskellige outputs. 

Jeg har vedhæftet de forskellige dokumenter med koden, men jeg har egentligt kun ændret en del af, hvordan input messegesne bruges i triggertext. 

Der er også vedhæftet en video-fil, som viser, hvordan programmet fungere. Har brugt Triggertext og mouseXY.
